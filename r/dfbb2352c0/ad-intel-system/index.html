SETUP GUIDE

Ad Intelligence System

The full pipeline for analyzing any brand's Facebook ads and producing a strategic brief organized as seven questions that hands directly to an AI agent as creative production context.

  • Scrapes active ads from the Facebook Ad Library, ranked by distribution
  • Extracts video frames at 10fps, transcribes audio locally, scrapes the top 2 landing pages
  • Hands the gathered data to Claude Code, which produces an HTML brief with the seven-question structure
  • Output is a single self-contained HTML file you copy-paste into your agent's context

This is the full version of the system shown in the YouTube video on competitive ad analysis. The lite version (single skill, no video frame analysis, no LP scrape) is also available.

01What you need before you start

Before anything, make sure you have the following. If you don't have something on this list, the rest won't work.

  • A computer running macOS or Windows. Apple Silicon Macs (M1/M2/M3/M4) get a major speed boost from a faster local Whisper. Intel Macs and Windows still work, just slower.
  • A Claude subscription (Pro or Team) — needed to run Claude Code.
  • ~5 GB of free disk space for the Whisper model + Playwright browser binaries.
  • About 90–120 minutes of focused time for the first setup.
💡 You DO NOT need

API keys for OpenAI, Anthropic, or Gemini. No Whisper API. No vision API. Everything analytical runs through Claude Code (which uses your subscription). Everything else (scraping, frame extraction, transcription, landing page scraping) runs locally on your machine.

02Install the prerequisites

You'll need five pieces of software:

  1. Python 3.11+
  2. FFmpeg — for extracting video frames and audio
  3. Playwright (Python library + Chromium browser) — for landing page scraping
  4. Whisper (mlx-whisper on Apple Silicon, openai-whisper everywhere else) — for audio transcription
  5. Claude Code — runs the skill

2.1 — Install Python

Same as the contribution-margin-system guide. Briefly:

Mac (Terminal)
python3 --version
Windows (Command Prompt)
python --version

If you see Python 3.11.x or higher, you're good. If not, install from python.org/downloads. On Windows, check the "Add python.exe to PATH" box on the first installer screen.

2.2 — Install FFmpeg

FFmpeg extracts video frames and audio. It's a separate program — not a Python library.

On Mac (using Homebrew)

If you don't have Homebrew, install it first from brew.sh. Then:

brew install ffmpeg

Confirm it works:

ffmpeg -version

On Windows

  1. Go to gyan.dev/ffmpeg/builds (the official recommended Windows builds).
  2. Download the "release essentials" zip (e.g. ffmpeg-release-essentials.zip).
  3. Extract to a folder you'll keep, e.g. C:\ffmpeg.
  4. Add C:\ffmpeg\bin to your PATH:
    • Press Windows key, type "environment variables", press Enter.
    • Click "Environment Variables" at the bottom.
    • Under "User variables", select Path, click Edit, click New, paste C:\ffmpeg\bin.
    • Click OK on every dialog to save.
  5. Close Command Prompt completely, reopen, and run ffmpeg -version to confirm.

2.3 — Install Python libraries

The bundle includes a requirements.txt file. After unzipping (section 3), navigate to the folder and run:

Mac
pip3 install -r requirements.txt
Windows
pip install -r requirements.txt

This installs requests and playwright. After that, install the actual Chromium browser Playwright uses:

playwright install chromium

2.4 — Install Whisper (audio transcription)

⚠ Pick the right one for your platform

Apple Silicon Macs (M1/M2/M3/M4): use mlx-whisper — runs ~5–10× faster on Apple's Neural Engine. Intel Macs, Windows, Linux: use openai-whisper. The transcribe-audio.py script auto-detects which one is installed.

Apple Silicon Macs

pip3 install mlx-whisper

Everywhere else (Intel Mac, Windows, Linux)

pip install openai-whisper

The first time you transcribe a video, Whisper will download the "base" model (~150 MB). This happens automatically — just wait for the first run to finish.

2.5 — Install Claude Code

Free tool from Anthropic. Follow the install instructions at docs.claude.com/en/docs/claude-code/quickstart and log in with your Claude account. Once you see the ❯ prompt, you're set.

04Install the skill into Claude Code

The skill is a markdown file Claude Code reads to know how to run the pipeline. Copy it into a special .claude/skills/ folder inside the project so Claude Code finds it.

  1. Open Terminal (Mac) or Command Prompt (Windows).
  2. Navigate to the project folder:
    • Mac: cd ~/Desktop/ad-intelligence-system
    • Windows: cd %USERPROFILE%\Desktop\ad-intelligence-system
  3. Create the .claude/skills/ directory and copy the skill into it:
    Mac
    mkdir -p .claude/skills
cp -r skills/ad-intel-brief .claude/skills/
    Windows
    mkdir .claude\skills
xcopy /E /I skills\ad-intel-brief .claude\skills\ad-intel-brief
  4. Start Claude Code: 
    claude
  5. Once you see ❯, the skill is loaded.

05Run your first brief

From inside Claude Code, just say:

Run an ad intel brief on [BRAND NAME]

Claude Code will handle everything from there — running each script in sequence, then synthesizing the analytical sections.

If you know the Facebook Page ID, include it for more precise results:

Run an ad intel brief on [BRAND NAME], page ID 123456789

What happens, in order

  1. Scrape ads (~4 min for ~175 ads) — pulls from the FB Ad Library API, sorted by impressions high to low. Saves JSON.
  2. Score every ad (instant) — 8 signals → A/B/C grades. The brief uses these to pick the top 10.
  3. Download top-10 media (~30 sec) — posters and videos. Note: CDN URLs expire within hours, so this happens immediately after scraping.
  4. Extract frames (~1 min) — 10fps for the first 15 seconds of each top-10 video. Produces ~150 frames per video — enough granularity to read the first 3 seconds (the hook).
  5. Transcribe audio (~10 min on Apple Silicon, longer elsewhere) — Whisper runs locally. First run downloads the base model.
  6. Identify + scrape top 2 landing pages (~30 sec each) — Playwright opens each LP, takes a full-page screenshot, and saves the HTML.
  7. Read avatar from copy (Q3) — Claude Code reads across all top-30 ads and the LPs.
  8. Extract the thesis (Q4) — Claude Code applies the six-card framework across the full corpus.
  9. Classify hook patterns (Q5) — Claude Code reads the frames + transcripts and ranks by which patterns sit highest in the API order.
  10. Synthesize the offer (Q6) — Claude Code reads the LP HTML/screenshot, extracts price/compare-at/guarantee/urgency/bonuses.
  11. DO vs DON'T (Q7) — Claude Code counts what the brand never does across the full ad library.
  12. Per-ad breakdown cards for the top 10 — Visual / Hook / Copy / Offer / Awareness / Hypothesis.
  13. Action templates — 4 copy-mode templates and 4 contrast-mode templates.
  14. Render HTML — single self-contained file at briefs/[brand-slug]-brief.html. All media embedded as base64 data URIs, so the file works without a server.
⚠ If you get 0 ads

Facebook rate-limited you. Wait 12–24 hours. Do not retry immediately — the limit lifts on its own, retrying makes it worse. A residential proxy ($5–10/mo) bypasses this permanently if it becomes a regular issue, but most users never need one.

06What the brief contains

The brief is a single HTML file organized around seven questions. The structure is fixed across brands — only the answers change.

Q1 What are they running, and how much on each?

  • Stats: active ad count, % video, oldest ad still running, copy repetition count, recent launches
  • Velocity callout: launches per day, scaling vs steady
  • Top 10 ads embedded inline as feed mockups with rank / days active / variant count / format annotations
  • Scoring methodology (collapsible, placed before the ads for context)

Q2 Where are they putting their money?

  • The 8-signal scoring methodology in detail
  • "Engagement lies" callout (why likes/comments/shares aren't signals)
  • Pattern across the top-scoring ads — what format, hook type, copy length, tone the brand keeps coming back to

Q3 Who is the avatar?

Reverse-engineered from copy — not from targeting data. Four mini-cards: Demographics, Identity, Attention, Turn-offs. Plus a plain-text avatar prompt formatted for pasting into another agent.

Q4 What do they believe? (The thesis)

Six mini-cards: Core belief, Contrast, The big idea, Unique mechanism, The villain, Strength assessment.

Q5 What hooks are working?

Hook patterns ranked by average impression rank, with brand examples and awareness-level mapping (which hooks serve which awareness stage).

Q6 What's the actual offer?

From the top-2 landing pages: price, compare-at, guarantee, urgency (or its absence), bonuses, and funnel awareness (problem-aware vs product-aware ad distribution).

Q7 What are they NOT doing?

DO vs DON'T two-column layout with counts. What a brand never does across hundreds of ads is a deliberate choice — it tells you who they're selling to.

Action: Copy mode

4 fill-in-the-blank templates extracted from the top hook patterns. Black background, teal text, bracketed variables.

Action: Contrast mode

4 contrarian hooks built from inverting the brand's dominant angles. NOT gap-filling — saying the OPPOSITE of their dominant message, then making it true.

07Using the brief with your agent

The brief is context for an agent — not a document to memorize. Once you have it, the workflow is:

  1. Open the brief HTML file in your browser.
  2. Click the floating "Copy brief for your agent" button at the bottom-right. The full brief text is now on your clipboard.
  3. Open Claude Code (or any LLM client). Paste the brief.
  4. Ask in plain English:
    • "Using this brief as context, write me 3 ads for [my product] in copy mode"
    • "Using this brief as context, write me 3 ads for [my product] in contrast mode"

Copy mode vs contrast mode

Copy mode: take the brand's proven frame — hook pattern, angle, tone — and apply it to your product. You're not copying the ad. You're using an approach already proven to work on this type of person.

Contrast mode: every brand in a category converges on the same angles over time. The brief shows you exactly what that looks like. You point your agent at the opposite — something that feels completely unlike everything else in the feed, but is still true and credible.

💡 Decision rule

Same type of buyer as the brand you studied → copy the frame. Real point of difference, or category has already converged on the same angles → contrast.

08If something goes wrong

"0 ads returned"

Rate limited by Facebook. Wait 12–24 hours. Don't retry immediately — it makes the limit longer.

ffmpeg: command not found

FFmpeg isn't installed or isn't on PATH. Go back to section 2.2.

ModuleNotFoundError: No module named 'whisper' / 'mlx_whisper'

You skipped section 2.4. On Apple Silicon: pip3 install mlx-whisper. Everywhere else: pip install openai-whisper.

playwright._impl._api_types.Error: Executable doesn't exist

Playwright is installed but Chromium isn't. Run playwright install chromium.

CDN URLs expired on media

Facebook video URLs only stay valid for a few hours after the scrape. Re-scrape and download in the same session — don't pause between steps 1 and 3.

Brand not found by name

Ask Claude Code: "Search the FB Ad Library for [brand name] and find the page ID." Then re-run with the ID.

Whisper transcription is taking forever

On Intel Macs and Windows without a GPU, the base Whisper model can take 30+ min for 10 videos. Either wait, or upgrade your hardware. If you're hitting this regularly and want help, message me.

The brief synthesis ran out of Claude Code context

If Claude Code hits its context window during synthesis, restart Claude Code, navigate to the project folder so the skill auto-loads, and ask: "Continue the brief — the data is already in scraped/, media/, frames/, and lps/. Skip to step 7."

I'm stuck on something not in this list

Open Claude Code in the project folder and describe the problem. Claude can read the scripts, look at error messages, and walk you through fixes interactively.
