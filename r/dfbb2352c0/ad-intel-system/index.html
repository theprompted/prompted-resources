01Get started in four steps

You need Claude Code already installed (you're a paid Claude subscriber). That's the only assumption. Claude Code installs everything else for you — Python, FFmpeg, Whisper, Playwright — automatically.

  1. Download the zip using the button above. Move it to your Desktop and double-click to unzip. You'll get a folder called ad-intelligence-system.
  2. Open Claude Code in that folder. In Terminal (Mac) or Command Prompt (Windows): cd ~/Desktop/ad-intelligence-system && claude Windows: cd %USERPROFILE%\Desktop\ad-intelligence-system && claude
  3. Tell Claude Code to set up the system. Once you see the ❯ prompt, paste: set up the ad intel system Claude Code will detect what's missing on your machine and install everything: Python (if needed), FFmpeg, Playwright, Whisper, and copy the skills into place. It tells you what it's doing at each step.
  4. Run your first brief. Once setup completes, paste: run an ad intel brief on [BRAND NAME] Replace [BRAND NAME] with whatever brand you want to analyze. Claude Code handles the whole pipeline. The first brief takes ~20–30 minutes (the first run downloads the Whisper model).
⚠ Heads up — password prompts

Claude Code may ask for your computer password once or twice during setup (when installing Homebrew or system packages). That's normal — type your password into the Terminal window when you see the prompt. You won't see characters as you type. Press Enter when done.

02If something goes wrong

Just describe what you saw to Claude Code. It can read its own scripts, parse error messages, and walk you through fixes interactively — that's the whole point.

The most common issue is 0 ads returned: Facebook rate-limited your IP. Wait 12–24 hours and try again. Don't retry immediately — it makes the rate limit longer.

If a brief takes a long time on Whisper transcription (Intel Mac or Windows without GPU), that's normal — wait it out. If it's been over an hour, message me.
